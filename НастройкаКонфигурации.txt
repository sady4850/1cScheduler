================================================================================

Создайте следующие объекты в Конфигураторе:

────────────────────────────────────────────────────────────────────────────────
1. СПРАВОЧНИК: ПереговорныеКомнаты
────────────────────────────────────────────────────────────────────────────────

   Имя:           ПереговорныеКомнаты
   Синоним:        Переговорные комнаты
   Длина наименования: 100

   Реквизиты:      нет (используем только стандартное Наименование)

   После создания справочника зайдите в режим 1С:Предприятие и создайте
   два элемента:
     - Переговорная1
     - Переговорная2

────────────────────────────────────────────────────────────────────────────────
2. ДОКУМЕНТ: БронированиеПереговорной
────────────────────────────────────────────────────────────────────────────────

   Имя:           БронированиеПереговорной
   Синоним:        Бронирование переговорной
   Проведение:     Разрешить (оперативное и неоперативное)

   Реквизиты:
   ┌────────────────┬─────────────────────────────────────────┬──────────┐
   │ Имя            │ Тип                                     │ Обязат.  │
   ├────────────────┼─────────────────────────────────────────┼──────────┤
   │ Комната        │ СправочникСсылка.ПереговорныеКомнаты    │ Да       │
   │ ДатаНачала     │ Дата (Состав: Дата и время)             │ Да       │
   │ ДатаОкончания  │ Дата (Состав: Дата и время)             │ Да       │
   │ Тема           │ Строка (200)                            │ Нет      │
   │ Организатор    │ Строка (100)                            │ Нет      │
   └────────────────┴─────────────────────────────────────────┴──────────┘

   Модуль объекта:
     Скопируйте код из файла МодульОбъекта_БронированиеПереговорной.bsl

   Форма документа (управляемая):
     Имя: ФормаДокумента
     Назначьте как основную форму объекта.

     Элементы формы (перетащите реквизиты объекта на форму):
       - Комната
       - ДатаНачала
       - ДатаОкончания
       - Тема
       - Организатор (Только просмотр = Истина)
       - Стандартные команды: ЗаписатьИЗакрыть, Записать, Провести

     Модуль формы:
       Скопируйте код из файла МодульФормыДокумента_БронированиеПереговорной.bsl

────────────────────────────────────────────────────────────────────────────────
3. ОБРАБОТКА: КалендарьПереговорных
────────────────────────────────────────────────────────────────────────────────

   Имя:           КалендарьПереговорных
   Синоним:        Календарь переговорных

   Форма (управляемая):
     Имя: Форма
     Назначьте как основную форму.

     Реквизиты формы:
     ┌────────────────┬─────────────────────────┐
     │ Имя            │ Тип                     │
     ├────────────────┼─────────────────────────┤
     │ ДатаКалендаря  │ Дата                    │
     │ ТабДок         │ ТабличныйДокумент       │
     └────────────────┴─────────────────────────┘

     Элементы формы (сверху вниз):
     ┌─────────────────────────────────────────────────────────────┐
     │ [Группа: ПанельНавигации] (обычная группа, горизонтальная)  │
     │   [Кнопка: Назад]   — команда формы КомандаНазад            │
     │   [Поле: ДатаКалендаря] — поле ввода даты                   │
     │   [Кнопка: Вперед]  — команда формы КомандаВперед           │
     │   [Кнопка: Сегодня] — команда формы КомандаСегодня          │
     ├─────────────────────────────────────────────────────────────┤
     │ [Поле: ТабДок] — поле табличного документа                  │
     │   Растягивать по вертикали = Истина                         │
     │   Растягивать по горизонтали = Истина                       │
     │   Событие: ОбработкаРасшифровки -> ТабДокОбработкаРасшифровки│
     └─────────────────────────────────────────────────────────────┘

     Команды формы:
       - КомандаНазад   (Синоним: "<", Действие: КомандаНазад)
       - КомандаВперед  (Синоним: ">", Действие: КомандаВперед)
       - КомандаСегодня (Синоним: "Сегодня", Действие: КомандаСегодня)

     Модуль формы:
       Скопируйте код из файла МодульФормы_КалендарьПереговорных.bsl

────────────────────────────────────────────────────────────────────────────────
4. ПОДСИСТЕМА (необязательно): Бронирование
────────────────────────────────────────────────────────────────────────────────

   Добавьте в подсистему:
     - Справочник.ПереговорныеКомнаты
     - Документ.БронированиеПереговорной
     - Обработка.КалендарьПереговорных

   Включите подсистему в интерфейс командного интерфейса.

================================================================================
  После настройки обновите конфигурацию БД (F7) и запустите в режиме Предприятие.
  Откройте обработку "Календарь переговорных" — это основной интерфейс.
