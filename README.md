# Календарь бронирования переговорных для 1С:Предприятие 8.3

Готовое решение для бронирования переговорных комнат в 1С:Предприятие 8.3 с визуальным календарем и контролем пересечений.

## Что это

Модуль позволяет сотрудникам бронировать переговорные комнаты через наглядный календарь с получасовой сеткой. Система автоматически проверяет конфликты и не даёт создать пересекающиеся брони.

## Возможности

- **Визуальный календарь** -- табличный документ с сеткой 09:00--19:30, шаг 30 минут
- **Навигация по дням** -- кнопки "Назад", "Вперёд", "Сегодня"
- **Быстрое бронирование** -- клик по свободному слоту открывает форму создания брони
- **Просмотр существующих** -- клик по занятому слоту открывает документ бронирования
- **Контроль пересечений** -- при проведении документа проверяется, не занято ли время
- **Автоматическое заполнение** -- организатор подставляется из имени пользователя, длительность по умолчанию 30 минут
- **Цветовая индикация** -- свободные слоты зелёные, занятые красные

## Структура объектов конфигурации

| Объект | Тип | Назначение |
|--------|-----|------------|
| `ПереговорныеКомнаты` | Справочник | Список переговорных комнат |
| `БронированиеПереговорной` | Документ | Бронь конкретной комнаты на время |
| `КалендарьПереговорных` | Обработка | Основной интерфейс -- визуальный календарь |

## Файлы

| Файл | Описание |
|------|----------|
| `НастройкаКонфигурации.txt` | Пошаговая инструкция по созданию объектов в Конфигураторе |
| `МодульОбъекта_БронированиеПереговорной.bsl` | Валидация и проверка пересечений при проведении |
| `МодульФормы_КалендарьПереговорных.bsl` | Построение календаря, навигация, обработка кликов |
| `МодульФормыДокумента_БронированиеПереговорной.bsl` | Автозаполнение и оповещение календаря после записи |

## Установка

1. Откройте конфигурацию в Конфигураторе 1С:Предприятие 8.3
2. Создайте объекты по инструкции из `НастройкаКонфигурации.txt`
3. Скопируйте код из `.bsl`-файлов в соответствующие модули
4. Обновите конфигурацию базы данных (F7)
5. В режиме Предприятие создайте элементы справочника `ПереговорныеКомнаты`
6. Откройте обработку "Календарь переговорных"

## Требования

- 1С:Предприятие 8.3 (управляемые формы)
- Режим совместимости: не ниже 8.3.10
